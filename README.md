# AVATAR-ANIMATION
This project is a simple interactive web app where you can control a 3D avatar using text commands. The idea was to explore how a system can take natural language input and turn it into visible actions.

What it does
* You type a command (e.g. “wave hello”, “walk forward”)
* The system interprets the input
* It selects a matching animation
* The 3D avatar performs the action
* A short explanation is displayed to show what happened

What it does
The system uses a simple rule-based approach:
 * It looks for keywords in the text
 * Matches them to an intent (e.g. greeting, movement, hazard)
 * Triggers a predefined animation

Controls
 * Type a command and click Interpret & Animate
 * Use the scene buttons to switch environments
 * Use playback controls (play, pause, loop, reset)

 Tech used
  * HTML
  * CSS
  * JavaScript
